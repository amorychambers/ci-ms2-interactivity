Privacy Policy

This privacy policy seeks to establish the terms of use of the Steam Web API as utilised by Lost in Library, henceforth, 'the application'.

  • Lost in Library is not endorsed by or affiliated with either Valve or Steam.
  • The application does not access or make use of non-public enduser data. It uses the Steam Web API's public methods to access public information. The application does not make any private data public or accessible.
  • The application does not store any of the user's Steam data beyond the session. The data is not shared with any third-party and is not used for marketing purposes.
  • The application is intended to provide a user with their own data on provision of their individual Steam ID. However, this agreement is on good faith only that the user only accesses their own public Steam Data. As the application is not able to access any private data, it does not expose any data that is not publicly available using Steam's services.
  • The application is not for profit and was created as an assessment project for a diploma provided by Code Institute.
